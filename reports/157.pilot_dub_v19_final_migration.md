# PILOT-Dub v19 Final Migration

## Summary

The public PILOT-Dub project has been migrated from a dual-generator layout:

```text
pilot-dub/base
pilot-dub/strong
```

to a single final project path:

```text
pilot-dub/final
```

The final public checkpoint is now:

```text
artifacts/PILOT-Dub/generator_epoch00.ckpt
```

which is copied from the current `Glo-Var v19` line.

## What Changed

- Removed public generator configs:
  - `configs/model/pilot-dub/base.yaml`
  - `configs/model/pilot-dub/strong.yaml`
- Added public final config:
  - `configs/model/pilot-dub/final.yaml`
- Synced v19 model support into:
  - `src/modeling/nets/voicecraft_dub.py`
  - `src/lightning/module.py`
- Added v19 internal config chain:
  - `configs/model/glo-var/v19.yaml`
  - `configs/optimizer/v19_finetune_from_glovar_v14_2.yaml`
  - `configs/v19_real_time_acoustic_interface_formal.yaml`
- Updated public runtime defaults:
  - `configs/evaluate/LRS3_Test_True.yaml`
  - `configs/inference/inference.yaml`
  - `scripts/run_lrs3_metrics.sh`
  - `scripts/run_lrs3_metrics_mini.sh`
- Replaced generator artifacts:
  - removed `generator_base_epoch00.ckpt`
  - removed `generator_strong_epoch00.ckpt`
  - added `generator_epoch00.ckpt`
  - added `generator_epoch01.ckpt`
- Updated user-facing docs:
  - `README.md`
  - `AGENTS.md`
  - `artifacts/PILOT-Dub/MANIFEST.md`

## Current Public Story

PILOT-Dub is now presented as:

```text
text + target video + reference audio
    -> planner hidden
    -> real-time acoustic interface prediction
    -> autoregressive dubbing generation
```

Historical scorer / reranking files remain in the repo only as archived artifacts and helper scripts. They are no longer the public final method path.

## Validation

Smoke validation completed:

```text
pilot-dub/final + generator_epoch00.ckpt
```

with a one-sample `LRS3 mini20` `evaluate_npy.py` run completed successfully on GPU 7.
